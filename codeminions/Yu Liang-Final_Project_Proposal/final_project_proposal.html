Final Project Proposal

By Yu(Monica) Liang

Project Description

For my final project (individual project), I would like to create a static webpage introducing traditional cuisine in my hometown (Shandong Cuisine) and localize it into Simplified Chinese and Traditional Chinese.

Tools and Resources

I will be using HTML, CSS and Javascript to create the original webpage with appropriate styling and interactive elements. Articles, links and photos on the webpage are contents from Wikipedia or other online resources. For the localization process, I will use Trados to translate the HTML file. If necessary, relevant termbases will be created to produce consistent translation.

Webpage Discription

This webpage consists of three parts:

  • Articles

    • Articles introducting Shandong Cuisine.

  • Photos

    • Photos taken by me or from online resources.

  • Interactive quizzes or games

    • Interactive elements created with Javascript to test what the readers have learned from reading the articles.

Steps for Localization

  • Use Trados to set up filters for the translation of the HTML file
  • Run a pseudo translation based on the filter
  • Translate the content of the webpage in Trados
  • Run QA process to ensure everything is successfully localizaed into the desired languages
